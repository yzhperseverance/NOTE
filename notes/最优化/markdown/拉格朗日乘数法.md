# 拉格朗日乘数法

​	用于将约束优化问题转化为无约束优化问题。==**（主要解决等式优化问题）**==

​	对于等式约束可以直接引入λ，对于不等式约束要先通过松弛变量转化为等式约束，然后再引入λ，最后对所有变量（x、λ）求导等于0，解出各个变量的值。
